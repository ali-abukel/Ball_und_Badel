🏓 Ball & Paddel (Java Edition)
 Ein klassisches Arcade-Spiel, entwickelt mit Java, das Logik, Grafik-Rendering und Kollisionserkennung kombiniert.
📖 Beschreibung
Dieses Projekt ist ein interaktives Spiel, bei dem ein Ball mit einem beweglichen Paddel gesteuert werden muss. Es wurde entwickelt, um die Grundlagen der objektorientierten Programmierung (OOP) und der grafischen Benutzeroberflächen in Java zu vertiefen.

🚀 Features
Echtzeit-Physik: Berechnung von Abprallwinkeln und Ballgeschwindigkeit.
Tastatursteuerung: Reaktionsschnelle Bewegung des Paddels.
Score-Tracking: Integrierte Punkteanzeige während des Spiels.
Clean Code: Strukturierter Aufbau nach Java-Best-Practices.

🛠️ Technologien & Skills
Programmiersprache: Java
Konzepte: Objektorientierung (OOP), Event-Handling, Graphics2D.
IDE:IntelliJ
